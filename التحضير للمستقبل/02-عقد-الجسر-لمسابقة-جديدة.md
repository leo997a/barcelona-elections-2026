# عقد الجسر لمسابقة جديدة

هذا العقد هو الحد الأدنى الذي يجب أن يخرجه أي جسر بيانات حتى نستفيد من قوالب المونديال مع مسابقات أخرى.

## تعريف المسابقة

```ts
type CompetitionProfile = {
  competitionId: string;
  seasonId: string;
  slug: string;
  nameAr: string;
  nameEn: string;
  type: 'league' | 'cup' | 'group_knockout';
  provider: 'fotmob' | 'reo-bridge' | 'manual';
  primaryColor: string;
  secondaryColor: string;
  logoUrl?: string;
};
```

أمثلة:

- دوري أبطال أوروبا: `type = group_knockout`
- الدوري الإسباني: `type = league`
- كأس آسيا: `type = group_knockout`

## ملخص البطولة

```ts
type CompetitionSnapshot = {
  competition: CompetitionProfile;
  fetchedAt: string;
  fixtures: CompetitionFixture[];
  standings?: CompetitionGroup[];
  bracket?: CompetitionBracket;
  topScorers?: CompetitionScorer[];
  teams: CompetitionTeam[];
};
```

## المباراة

```ts
type CompetitionFixture = {
  id: string;
  stage?: string;
  group?: string;
  date: string;
  status: 'scheduled' | 'live' | 'half-time' | 'finished' | 'postponed';
  minute?: string | number;
  home: CompetitionTeam;
  away: CompetitionTeam;
  homeScore?: number;
  awayScore?: number;
  venue?: string;
};
```

## تفاصيل المباراة

```ts
type CompetitionMatchDetails = {
  match: CompetitionFixture;
  events: Array<{
    minute: number;
    type: 'goal' | 'yellow' | 'red' | 'substitution' | 'var' | string;
    player: string;
    team: 'home' | 'away' | 'neutral';
  }>;
  lineups: {
    home?: CompetitionLineup;
    away?: CompetitionLineup;
  };
  teamStats: Array<{
    key: string;
    label: string;
    home: number | string;
    away: number | string;
  }>;
  playerOfTheMatch?: CompetitionPlayer;
};
```

## التحويل إلى حقول القوالب

أي جسر جديد يجب أن يمر عبر محول واحد يحول بيانات المسابقة إلى هذه الحقول:

| حقل القالب | مصدره |
|---|---|
| `homeTeam`, `awayTeam` | أسماء الفرق |
| `homeCode`, `awayCode` | كود دولة أو اختصار فريق |
| `homeLogo`, `awayLogo` | شعار الفريق |
| `homeScore`, `awayScore` | النتيجة |
| `matchStatus`, `minute` | حالة المباراة |
| `matchVenue`, `matchDate`, `matchTime` | معلومات المباراة |
| `playersJson` | تشكيلة الفريق المختار |
| `eventsJson` | أحداث المباراة |
| `statsJson` | إحصائيات مختصرة |
| `groupsJson` | ترتيب المجموعات أو جدول الدوري |
| `scorersJson` | الهدافون |
| `bracketJson` | الأدوار الإقصائية |

## حقول الإحصائيات القياسية

هذه الحقول يجب أن تكون الهدف الموحد لكل مسابقة:

- `statPossessionHome/Away`
- `statXgHome/Away`
- `statShotsHome/Away`
- `statOnTargetHome/Away`
- `statShotAccuracyHome/Away`
- `statPressureHome/Away`
- `statFieldTiltHome/Away`
- `statRecoveriesHome/Away`
- `statDuelsHome/Away`
- `statCornersHome/Away`
- `statFoulsHome/Away`
- `statYellowHome/Away`
- `statPassHome/Away`

إذا لم يوفر المصدر قيمة معينة، نستخدم قيمة يدوية أو مؤشر مركب محسوب من قيم أخرى.

## قاعدة مهمة

لا نجعل القوالب تعرف اسم المصدر. القالب يعرف فقط الحقول الموحدة. الجسر هو المسؤول عن فهم FotMob أو أي مصدر آخر.
