# مصادر اللاعبين للبطولات القادمة

## الفكرة

أي بطولة مستقبلية لا يجب أن تحتاج إلى إعادة بناء قوالب اللاعبين من الصفر. المطلوب هو أن يوفر الجسر نفس العقد الأساسي، ثم تستخدم القوالب نفس منطق الاختيار.

## المصادر التي يجب دعمها في أي بطولة

| المصدر | الاستخدام |
|---|---|
| رجل المباراة | بطاقة ما بعد المباراة أو لحظة إعلان الأفضل |
| أفضل لاعبي المضيف | تحليل مباشر للطرف الأول |
| أفضل لاعبي الضيف | تحليل مباشر للطرف الثاني |
| لاعب من تشكيلة المضيف | تقديم التشكيلة أو لاعب تحت الضوء |
| لاعب من تشكيلة الضيف | تقديم التشكيلة أو لاعب تحت الضوء |
| بيانات يدوية | عند غياب البيانات أو للتخصيص التحريري |

## أنواع الإحصائيات

| النوع | أمثلة البيانات |
|---|---|
| هجوم | أهداف، تسديد، xG، فرص، صناعة |
| تمرير | تمريرات، تمريرات دقيقة، كرات مفتاحية، عرضيات |
| دفاع | التحامات، افتكاكات، اعتراضات، استرداد، تشتيت |
| تلقائي | أفضل المتاح من الجسر |

## كيف نستفيد منه لاحقاً

عند إضافة دوري الأبطال أو الدوري الإسباني، نحتاج فقط إلى:

1. جسر يعيد `playerOfTheMatch`.
2. قوائم `topPlayers.home` و`topPlayers.away`.
3. تشكيلات قابلة للتحويل إلى لاعبين عبر نفس دالة التشكيلة.
4. إحصائيات اللاعب بمفاتيح قابلة للفلترة مثل `goal` و`pass` و`tackle`.

بهذا تصبح القوالب نفسها قابلة لإعادة الاستخدام مع تغيير مصدر البطولة فقط.
