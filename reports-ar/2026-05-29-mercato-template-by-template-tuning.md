# تقرير عربي: ضبط قوالب الميركاتو قالبًا قالبًا

التاريخ: 2026-05-29  
اسم المرحلة: MERCATO-TEMPLATE-BY-TEMPLATE-TUNING-001  
النطاق: ضبط محدود وآمن لقوالب الميركاتو داخل `constants.ts` فقط، مع تقرير عربي جديد.

## 1. ملخص تنفيذي

هذه المرحلة جاءت بعد ملاحظة أن بعض أفكار القوالب، رغم أنها تعمل تقنيًا، ما زالت تحتاج لغة بث أفخم وتناسقًا أوضح داخل المكتبة والإعدادات. تم التركيز على قوالب الميركاتو، خصوصًا القوالب التي كانت ما زالت تظهر بأسماء أو حقول إنجليزية مثل:

- `Mercato - Player Season Card`
- `Mercato - Player Impact Forecast`
- `Mercato - Contract Exit Watch`
- `Mercato - Here We Go Board`
- `ميركاتو — Deadline Day`

النتيجة: أسماء قوالب الميركاتو القديمة والجديدة أصبحت عربية ومتناسقة، والحقول الأساسية داخل الإعدادات أصبحت عربية، مع تحسين القيم الافتراضية لتبدو كقوالب بث قابلة للتحديث بدل نصوص إنجليزية جامدة.

لم يتم تغيير أي `templateId`.  
لم يتم تعديل renderers أو API أو Player Intel أو Stream Deck.  
لم يتم إغلاق العرض المباشر أو المساس برابط OBS.

## 2. الملفات المعدلة

   - ضبط قيم افتراضية عربية للحقول المهمة.

2. `reports-ar/2026-05-29-mercato-template-by-template-tuning.md`

## 3. ما تم ضبطه في Mercato Media

قوالب Mercato Media العشرة السابقة بقيت بنفس IDs، لكن أصبح لكل قالب ضبط مستقل بدل الاعتماد الكامل على نفس القيم العامة:

| القالب | الضبط |
|---|---|
| مجلس الصفقة الملكي | تعتيم متوسط، panel أوضح، موسيقى أخف |
| أطلس الظل للمفاوضات | تعتيم أعلى، blur بسيط، سطوع أقل |
| ختم العقد السري | سطوع أعلى، panel أقوى، تعتيم أخف |
| بيان الطائرة الخاصة | توازن بين الصورة والنص مع تعتيم متوسط |
| غرفة الاتصال المشفّر | blur أعلى وسطوع أقل لجو اتصالات سري |
| عملية خطف الصفقة | تعتيم قوي وموسيقى أعلى نسبيًا |
| بورصة نبض الجماهير | سطوع أعلى ونص أصغر قليلًا لاستيعاب التفاصيل |
| مختبر الضوء الأخضر | موسيقى منخفضة وسطح بصري أنظف |
| خزنة التمويل الذهبية | panel قوي حتى لا تضيع أرقام التمويل |
| مركز قيادة اليوم الأخير | تعتيم أعلى ونص أصغر قليلًا لكثافة اليوم الأخير |

## 4. ما تم ضبطه قالبًا قالبًا في القوالب القديمة

### 4.1 إعلان صفقة سينمائي

- الاسم أصبح `ميركاتو — إعلان صفقة سينمائي`.
- `designStyle` الظاهر تحوّل إلى خيارات عربية بدل أكواد تقنية.
- `visualVariant` أصبح عربيًا: زجاج نيون، تكتيكي داكن، استوديو فاخر، بث نظيف.
- تم تعريب حقول Sportmonks، الفريق، المركز، الشعارات، عنوان السوق، عناصر JSON، الأخبار، الصفقات المتوقعة.

### 4.2 غرفة القرار الذكية

- الاسم أصبح `ميركاتو — غرفة القرار الذكية`.
- العنوان الافتراضي أصبح `غرفة قرار الميركاتو`.
- تم تعريب اللاعب الرئيسي، النادي الحالي، النادي الوجهة، قيمة الصفقة، المصدر، JSON السوق، الألوان، الحركة، وحقول الموضع.

### 4.3 مصفوفة توقعات الانتقال

- الاسم أصبح `ميركاتو — مصفوفة توقعات الانتقال`.
- العنوان الافتراضي أصبح `مصفوفة الاحتمالات`.
- تم تعريب عناصر الاحتمالات ودرجات الحرارة مثل ساخن، مراقبة، متوقع، صعب، بارد.

### 4.4 صفقات تمت اليوم

- الاسم أصبح `ميركاتو — صفقات تمت اليوم`.
- العنوان الافتراضي أصبح `صفقات حُسمت اليوم`.
- تم تعريب الصفقة الرئيسية، النادي الحالي، النادي الوجهة، المصدر، JSON الصفقات المحسومة، وحقول الألوان والموضع.

### 4.5 بطاقة موسم اللاعب

- الاسم أصبح `ميركاتو — بطاقة موسم اللاعب`.
- الوصف أصبح عربيًا وموجهًا للبث.
- تم تعريب كل حقول اللاعب، الصورة، Sportmonks، الموسم، النادي، الثقة، المصدر، الإحصائيات، قراءة السوق، الصفقات اليومية، JSON السوق.

### 4.6 توقع تأثير اللاعب

- الاسم أصبح `ميركاتو — توقع تأثير اللاعب`.
- العنوان الافتراضي أصبح `توقع التأثير`.
- تم تعريب مؤشرات الأداء مثل المشاركات، الدقائق، التمريرات التقدمية، الدقة، والطاقة.

### 4.7 مراقبة نهاية العقد

- الاسم أصبح `ميركاتو — مراقبة نهاية العقد`.
- العنوان الافتراضي أصبح `مراقبة نهاية العقد`.
- تم تعريب سيناريو الانتقال الحر، احتمالية الخروج، قراءة السوق، وبيانات JSON.

### 4.8 لوحة الحسم النهائي

- الاسم أصبح `ميركاتو — لوحة الحسم النهائي`.
- العنوان الافتراضي أصبح `لوحة الحسم النهائي`.
- تم تعريب الصفقة البارزة، نسبة الثقة، آخر الأخبار، الصفقات اليومية، وعناصر JSON.

### 4.9 اليوم الأخير

- الاسم أصبح `ميركاتو — اليوم الأخير`.
- تم إزالة `Deadline Day` من اسم القالب الظاهر.
- تم تعريب خيارات قديمة كانت تعرض `Deadline Day` داخل بعض الثيمات.
