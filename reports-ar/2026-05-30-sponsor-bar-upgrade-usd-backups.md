# تقرير مرحلة: تطوير شريط الداعمين والتحويل للدولار والنسخ الاحتياطية

التاريخ: 2026-05-30  
اسم المرحلة: SPONSOR-BAR-UPGRADE-USD-BACKUPS-001  
نوع المرحلة: تحسين قالب موجود + تحسين إدارة بيانات الداعمين

## 1. الهدف

كان المطلوب تطوير قالب `شريط الداعمين — شريط البث` ليصبح أقوى بصريًا وأسهل في الإدارة أثناء البث، مع:

- توسعة مبالغ الدعم والـ quick top-up.
- تحويل أسرع إلى الدولار.
- ميزات عرض جديدة داخل الشريط.
- تصدير واستيراد نسخ احتياطية للداعمين.

تم تنفيذ المرحلة بدون إضافة API جديد وبدون تغيير template ID.

## 2. الملفات التي تغيّرت

- `components/renderers/LeaderboardRenderer.tsx`
- `pages/Editor.tsx`
- `services/currencyService.ts`
- `constants.ts`
- `reports-ar/2026-05-30-sponsor-bar-upgrade-usd-backups.md`

لم يتم تعديل:

- `api/`
- `services/audioEngine.ts`
- Player Intel
- Stream Deck
- أسرار أو `.env`
- أي template ID قديم

## 3. تطوير شريط الداعمين بصريًا

تمت إعادة بناء `LeaderboardRenderer` ليصبح أقرب إلى شريط بث حديث:

- Header أقوى للشريط.
- إحصائيات أعلى الشريط:
  - إجمالي الدعم بالدولار.
  - عدد الداعمين.
  - عدد الدفعات.
- هدف دعم بالدولار مع شريط تقدم.
- أعلى داعم في أسفل الشريط.
- كل داعم يظهر معه:
  - الترتيب.
  - الاسم.
  - الصورة أو Avatar تلقائي.
  - إجمالي الدعم بالدولار.
  - المبلغ الأصلي والعملـة.
  - آخر دفعة.
  - نسبة مساهمته من إجمالي الدعم.

كما تمت إضافة وضع عرض جديد:

```text
sponsorDisplayMode = elite_wall
```

ووضع مدمج:

```text
sponsorDisplayMode = compact_stack
```

## 4. حقول جديدة للقالب

تمت إضافة حقول إعدادات جديدة إلى قالب شريط الداعمين:

- `showSponsorStats`
- `showGoalProgress`
- `fundraisingGoalUsd`
- `sponsorDisplayMode`

الحقول الجديدة لا تكسر البيانات القديمة. إذا لم تكن موجودة في instance قديم، يستخدم renderer قيمًا افتراضية آمنة.

## 5. التحويل إلى الدولار

تم تحسين `currencyService`:

- دعم cache الحالي كما هو.
- إضافة timeout للشبكة حتى لا يعلق الإدخال أثناء البث.
- إضافة أسعار fallback محلية لأهم العملات.
- دعم عملات إضافية:
  - MAD
  - DZD
  - TND
  - TRY
  - BRL
  - MXN
  - INR
  - IDR
  - MYR
  - SGD
  - CNY
  - KRW

النتيجة: عند إدخال مبلغ جديد يمكن إظهار تحويل USD بسرعة حتى لو الشبكة بطيئة أو أسعار العملات لم تُحمّل بعد.

## 6. تطوير غرفة إدارة الداعمين

تم تحسين تبويب `Sponsors` في Editor:

- إحصائيات مباشرة:
  - إجمالي الدولار.
  - عدد الداعمين.
  - عدد التبرعات.
  - اسم أعلى داعم.
- زر `تحديث USD` لإعادة احتساب كل المبالغ حسب العملة.
- مبالغ سريعة موسعة:
  - 5
  - 10
  - 25
  - 50
  - 100
  - 250
  - 500
  - 1000
  - 5000
  - 10000
- عرض أوضح لقائمة الداعمين.
- عرض تاريخ الدفعات مع USD لكل دفعة.
- رسائل نجاح/خطأ بعد عمليات النسخ أو التحويل أو الإضافة.

## 7. النسخ الاحتياطية

تمت إضافة:

- `تصدير نسخة`
- `استيراد نسخة`

التصدير ينشئ ملف JSON يحتوي:

- version
- exportedAt
- overlayId
- overlayName
- headline
- sponsors

الاستيراد يقبل:

- ملف يحتوي array مباشرة.
- أو ملف يحتوي `{ sponsors: [...] }`.

بعد الاستيراد يتم:

- تنظيف البيانات الأساسية.
- إعادة احتساب USD لكل داعم.
- ترتيب الداعمين حسب إجمالي الدولار.

## 8. التحقق البصري

تم تشغيل نسخة محلية على:

```text
http://127.0.0.1:5184
```

وتم التقاط لقطة Chrome headless بدقة 1280x720 لشريط الداعمين ببيانات اختبار محلية غير ملتزمة في Git.

النتيجة:

- الشريط يظهر.
- الإحصائيات تظهر.
- هدف الدعم يظهر.
- قائمة الداعمين تظهر.
- لا توجد صورة بيضاء بسبب كسر renderer.

ملاحظة: الخلفية البيضاء في لقطة Chrome headless كانت بسبب شفافية صفحة output وليست جزءًا من القالب.

## 10. المخاطر

الخطر متوسط لأن `LeaderboardRenderer` وواجهة Sponsors في Editor تغيّرا بشكل واضح، لكن الخطر مضبوط لأن:

- لا توجد API جديدة.
- لا يوجد تغيير في template ID.
- البيانات القديمة في `sponsorsData` تبقى مقروءة.
- الحقول الجديدة لها defaults.
- النسخ الاحتياطية JSON محلية في المتصفح.

## 11. QA يدوي مقترح

1. افتح قالب شريط الداعمين من Library.
2. ادخل تبويب Sponsors.
3. أضف داعمًا بعملة SAR أو EUR أو KWD.
4. تأكد أن USD يظهر فورًا.
5. استخدم quick top-up بقيم 5000 أو 10000.
6. اضغط `تحديث USD`.
7. اضغط `تصدير نسخة`.
8. احذف داعمًا تجريبيًا.
9. اضغط `استيراد نسخة` وأعد الملف.
10. افتح رابط العرض في OBS وتأكد أن الشريط يظهر بشكل صحيح.

## 12. الخلاصة

تم تحويل شريط الداعمين من قائمة بسيطة إلى لوحة دعم بث أكثر احترافية، مع تحويل أسرع للدولار، مبالغ كبيرة، هدف دعم، إحصائيات، وأدوات export/import للنسخ الاحتياطية. المرحلة جاهزة للرفع.
