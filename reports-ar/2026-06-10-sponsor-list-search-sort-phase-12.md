# تقرير عربي: SPONSOR-LIST-SEARCH-SORT-012

التاريخ: 2026-06-10
النطاق: تحسين صغير ومحدود لقائمة الداعمين داخل `Editor.tsx` فقط.

## 1. ملخص تنفيذي

هذه المرحلة تكمل مسار تنظيم قالب الداعمين بعد توسيع العملات وإضافة البحث داخل اختيار العملة. المشكلة المتبقية كانت أن قائمة الداعمين نفسها تكبر بسرعة، ولا يوجد فيها بحث أو ترتيب عرض سريع. الزر القديم `ترتيب` كان يرتب البيانات فعليًا حسب الدولار فقط، وهذا مفيد للحفظ، لكنه ليس كافيًا أثناء الإدارة اليومية عندما يريد المستخدم العثور على داعم باسم، دولة، عملة، أو قيمة.

تمت إضافة بحث وترتيب عرض داخل قائمة الداعمين بدون تغيير شكل بيانات القالب، وبدون تعديل الرندر، وبدون إضافة API أو functions جديدة.

## 2. التشخيص قبل التعديل

- قائمة الداعمين كانت ترسم `parseSponsors()` مباشرة.
- لا يوجد بحث داخل القائمة بعد إضافة عدد كبير من الدول والعملات.
- زر `ترتيب` الحالي يغير ترتيب البيانات المحفوظة، ولا يوفر أوضاع عرض مؤقتة.
- الوصول إلى داعم معيّن يصبح بطيئًا عند كثرة الإدخالات.
- لا توجد رسالة عندما لا تطابق نتائج البحث أي داعم.

## 3. التعديل المنفذ

تم تعديل `pages/Editor.tsx` فقط:

- إضافة حالة `sponsorListSearch` لبحث قائمة الداعمين.
- إضافة حالة `sponsorListSortMode` لاختيار ترتيب العرض.
- إضافة خيارات ترتيب عرض:
  - الدولار.
  - الأحدث.
  - الاسم.
  - الدولة.
- إضافة مساعد `getSponsorCountryLabel()` لعرض اسم الدولة باستقرار.
- إضافة مساعد `getSponsorLatestDonationAt()` لترتيب الداعمين حسب آخر دفعة.
- إضافة مساعد `getVisibleSponsors()` لتطبيق البحث والترتيب على نسخة عرض فقط.
- إضافة حقل بحث أعلى القائمة.
- إضافة أزرار ترتيب صغيرة وثابتة المقاس.
- إضافة رسالة `لا توجد نتائج مطابقة.` عندما لا يعثر البحث على داعمين.

## 6. قائمة QA يدوية

عند الدخول بمفتاح صالح:

1. افتح قالب الداعمين في Library.
2. اذهب إلى تبويب البيانات.
3. أضف 5 داعمين بأسماء وعملات مختلفة.
4. ابحث باسم داعم وتأكد أن القائمة تصفي النتائج.
5. ابحث باسم دولة أو رمز عملة مثل `SAR` أو `العراق`.
6. جرّب ترتيب الدولار، الأحدث، الاسم، الدولة.
7. اضغط زر `ترتيب` القديم وتأكد أنه لا يزال يرتب البيانات المحفوظة حسب الدولار.
8. افتح سجل داعم بعد البحث وتأكد أنه يظهر لنفس الداعم.
9. أضف مبلغًا لداعم من نتيجة بحث وتأكد أن الإجمالي يتحدث.
10. امسح البحث وتأكد أن كل الداعمين يعودون للظهور.
