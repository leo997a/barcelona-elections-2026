# تقرير الدفعة 31 - نقل حقول البيانات الخام إلى المتقدم

التاريخ: 2026-06-27

## الهدف

تحسين إعدادات كل القوالب بحيث لا تختلط حقول JSON وحقول الجسور والربط المباشر مع إعدادات التصميم والستايل. هذا مهم خصوصًا لقوالب الهدافين، الإحصائيات، المجموعات، الإقصائيات، وتصريحات المصادر.

## ما تم تنفيذه

- أي حقل ينتهي بـ `Json` أصبح يصنف تلقائيًا ضمن الإعدادات المتقدمة.
- حقول جسر إحصائيات اللاعبين مثل `playerStatsDataMode` و`playerStatsApiUrl` و`playerStatsPollSec` أصبحت ضمن المتقدم.
- حقول اختيار أقسام جلب إحصائيات اللاعب `include...` أصبحت ضمن المتقدم لأنها تخص البيانات لا التصميم.
- وضع قالب إحصائيات اللاعب `playerStatsMode` أصبح ضمن العرض لأنه يغير شكل القالب وسلوكه البصري.
- حقول روابط الصوت المخصصة بقيت ضمن الصوت حتى لا تنتقل خطأ إلى المتقدم.
- أضيف اختبار يحمي هذا التصنيف من الرجوع لاحقًا.

## أثر التغيير

- قالب الهدافين سيعرض إعدادات الستايل قبل JSON الهدافين اليدوي.
- قوالب اللاعب والإحصائيات ستكون أسهل في الضبط، لأن الجسر والـ API والتحديث لا تظهر في وسط الإعدادات البصرية.
- هذا يجعل إضافة قوالب جديدة أكثر استقرارًا لأن أي JSON جديد سينتقل تلقائيًا إلى المتقدم.
